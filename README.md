# SAP Data Analytics Project

## Overview
This project analyzes sales, procurement, and financial data to generate business insights.

## Analysis Performed
- Sales Analysis (Product-wise revenue)
- Financial Analysis (Revenue trend & profit distribution)
- Procurement Analysis (Vendor cost comparison)

## Tools Used
- SAP Concepts (SD, MM, FI)
- Excel / Google Sheets

## Key Insights
- Laptops generate highest revenue
- Business shows steady growth
- Vendor C is cost-effective
